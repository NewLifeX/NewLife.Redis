## 1. 产品概述

**NewLife.Redis** 是新生命团队自主研发的高性能、零外部重量级依赖的 .NET Redis 客户端组件，核心定位：

- 覆盖 Redis 主流版本（2.x → 7.x）的协议与命令
- 提供生产级可靠消息队列封装
- 支持公有云 Redis 托管服务（阿里云 / 腾讯云 / 华为云）的特殊版本与认证方式
- 与新生命 .NET 生态（日志、追踪、配置、序列化）无缝融合
- 自 2017 年起，已在日均 **80+ 亿次命令调用** 的生产环境稳定运行

**核心价值主张：**
- 高吞吐：连接池 + 同步高效 RESP 协议解析 + 可选自动管道合并
- 易集成：一个 NuGet 包，支持 .NET 4.5 ～ .NET 10+，配置简单
- 可靠消费：多种消息队列范式（简单队列 / 可靠队列 / 延迟队列 / Stream 流式消费）
- 云原生：适配三大主流公有云 Redis 托管服务特殊机制

## 3. 术语定义

| 术语 | 含义 |
|------|------|
| RESP | Redis Serialization Protocol，Redis 序列化协议 |
| RESP2 | Redis 2.0 引入的经典协议格式（简单字符串/错误/整数/批量字符串/数组） |
| RESP3 | Redis 6.0 引入的增强协议格式（Map/Set/Double/BigNumber/BlobError/Attribute） |
| Pipeline | 管道模式，批量发送命令无需等待每条响应 |
| Cluster | Redis Cluster，官方原生分片集群 |
| Sentinel | Redis Sentinel，官方主从哨兵高可用方案 |
| Replication | Redis 主从复制 |
| ACL | Redis 6.0 引入的访问控制列表 |
| Garnet | 微软开源的 Redis 协议兼容高性能实现 |
| Tair | 阿里云自研的 Redis 兼容存储引擎，提供额外数据结构 |
| TLS/SSL | 传输层安全，用于加密 Redis 通信 |
